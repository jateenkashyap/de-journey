# de-journey 🚀
A 6-month hands-on data engineering learning journey — built in public.

---

## 🔨 What I'm Building
AA structured 6-month hands-on journey to learn data engineering from scratch — covering Python, SQL, cloud storage, and pipeline orchestration. Each month builds on the previous one, with the goal of developing real, employable data engineering skills that work alongside modern AI tools.

---

## 🛠️ Tech Stack

| Month | Focus | Tools |
|-------|-------|-------|
| Month 1 | Dev Environment | VS Code, Git, GitHub, Python |
| Month 2 | Data Ingestion | Python, APIs, requests |
| Month 3 | Databases | SQL, PostgreSQL |
| Month 4 | Pipelines | Airflow / Prefect |
| Month 5 | Cloud | AWS / GCP |
| Month 6 | Capstone | Everything above |

## ▶️ How to Run

### 1. Clone the repo
```cmd
git clone git@github.com:yourusername/de-journey.git
cd de-journey
```

### 2. Set up the virtual environment
```cmd
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### 3. Run the script
```cmd
python hello.py
```
